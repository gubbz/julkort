## Git workflow
Create new branch for feature:
`git checkout -b <branch_name>`

When done:
1. `git add .`

2. `git commit -m 'Commit message'`

3. `git checkout master`
4. `git pull`
5. `git merge <branch_name>`
6. `git push`
