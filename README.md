# JulkortsApp 

## Create component:
1. Create a new class in screens, see Home.js for reference
2. Include class in router in App.js

## Git workflow
Create new branch for feature:
`git checkout -b <branch_name>`

When done:
`git add .`
`git commit -m 'Commit message'`

`git checkout master`
`git pull`
`git merge <branch_name>`
`git push`
